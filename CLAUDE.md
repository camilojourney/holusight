# Holusight

Public-facing product landing page for Holus — federated AI orchestrator.

## Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- pnpm

## Commands
- `pnpm dev` — start dev server
- `pnpm build` — production build
- `pnpm lint` — ESLint

## Design
- Dark theme, indigo/violet accents
- Enterprise-grade aesthetic
- See app/page.tsx for design system
